Privacy Policy

Last updated: August 18, 2022

Please read this Privacy Policy carefully before using JailBot. This Privacy Policy is subject to change at any time, and it is suggested you check it often.

Definitions

For the purposes of this Privacy Policy:

  • Text channel means a Discord text channel.
  • User means a Discord user.
  • User role means a Discord user role.
  • Guild means a Discord guild/server.

Information Collected

Per Guild:

  • ID of the guild.
  • Name of the guild.
  • ID of the jail category that is created.
  • ID of the text channel "tickets" that is created.
  • ID of the text channel "dispute-channel" that is created (if the option for it to be enabled is selected).
  • ID of the text channel "staff-commands" that is created.
  • ID of the text channel that is given by a guild administrator where notifications are sent of who was jailed and for how long. Also known as the "general channel" or "jail notifications channel".
  • ID of the user role "Jail" that is created.
  • ID of the user role "JailMod" that is created.
  • Timezone that is given by a guild administrator. This is encrypted when being stored.

Per User that is "Jailed":

  • ID of the guild they were jailed in.
  • ID of the user that was jailed.
  • ID of the "ticket" message that is sent in the "tickets" text channel.
  • ID of the user that jailed the user getting jailed.
  • The date and time the user is jailed on including year, month, day, hour, minute, and second.
  • The reason the user was jailed which is given by the user that has the role "Jail Mod" and is using the "/jail <member>" command to jail given user. This is message content and therefore encrypted when being stored as well as automatically deleted after 30 days as required by Discord.
  • Each user role the user being jailed has. This includes the ID of the guild the user role is in, ID of the user being jailed, and the name of the role.

How Collected Information is Used

Collected information is used solely for the functionality of the bot. Information from users that have been jailed is deleted upon release. Guild information is saved indefinitely.

Request Deletion of Information

Send an email to jailbot.support@protonmail.com to request deletion of your guild's saved information. Please include the ID of your guild. Get this by right-clicking on your server then selecting "Copy ID". You will then be asked to verify your data deletion request via Discord DMs to ensure you are the server owner.
